# DomX CRM Releases

See the full update and release guide at [**UPDATES.md**](../UPDATES.md) in the repo root.

Quick release:

```bash
# 1. Bump frontend/package.json version, commit, push
# 2. Tag and push
git tag v1.0.1
git push origin v1.0.1
# 3. Download domx-crm-release artifact from GitHub Actions
# 4. Upload all files to https://domx.low7labs.cloud/crm-updates/
```
